# ICT Course — Learning Path

Structured, progressive curriculum built from the MrWitness / AXL mentorship. Each module maps to the core curriculum volumes, organized so foundations come before execution.

> **KB vs. Course:** The `concepts/business-logic/` pages are reference material — dense and non-linear. These course pages are learner-facing: concept-first, with worked examples and homework extracted from transcripts. Use both together.

---

## Learning Path

```
Module 1 — Foundations
  01. What Moves the Market (Liquidity & Inefficiency)
  02. Fair Value Gaps (FVG / BISI / SIBI)
  03. Homework & Practice

Module 2 — Price Delivery
  01. Four Stages of Algorithmic Price Delivery
  02. Consolidation Model
  03. Expansion & Retracement Model
  04. Reversals (Three Types)

Module 3 — Session Context & Bias
  01. Power of Three (AMD)
  02. Session Kill Zones & Opening Prices
  03. Deviations (Fibonacci Targeting)
  04. Daily Bias

Module 4 — Market Structure
  01. Swing Classification (STH/ITH/LTH)
  02. Market Structure Fractality
  03. Structure Deviations (Two-Set Targeting)
  04. Model 2022, OTE, CSD

Module 5 — Order Flow & SMT
  01. HTF/LTF Order Flow
  02. SMT Divergence
```

---

## Prerequisite Order

You can begin at Module 1 regardless of experience level. Work through modules in order — each concept builds on the last:

1. **Module 1** unlocks everything else. Liquidity = where price goes; inefficiency = where price enters.
2. **Module 2** shows the four stages that structure *every* price run. Do not skip Stage 1 (consolidation).
3. **Module 3** adds time and session context — same concepts, new lens.
4. **Module 4** introduces multi-timeframe structure and the Model 2022 entry.
5. **Module 5** is the highest-timeframe filter (order flow) and an intermarket confluence tool (SMT).

After completing all five modules, work through the **Entry Models Library** (`concepts/entry-models/`) for the complete per-strategy checklists used in live trading.

## How to Use These Pages

Each lesson page has:
- **Concept** — the core idea, explained plainly
- **Rules** — the explicit trading rules the instructor stated
- **Worked Example** — a real trade or chart walk-through from the transcripts
- **Common Mistakes** — mistakes called out explicitly by MrWitness or AXL
- **Homework** — the assignment from that class session

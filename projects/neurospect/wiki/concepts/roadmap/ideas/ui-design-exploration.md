# UI Design Exploration ("Clause" Design)

Explore differentiated UI/workflow patterns. The current app uses shadcn/Radix/Tailwind primitives — production-quality but not distinctive. Explore a unique structural metaphor or design system that makes Neurospect immediately recognisable.

## Why it matters

Vertical AI products win on workflow density, not feature count. A signature UI is a moat against generic competitors. Worth running in parallel with feature work — it doesn't gate other tracks but informs every later interface decision.

## Dependencies

- None (parallel exploration).

## Open questions

- What's the "Clause" metaphor concretely — narrative timeline? trade-as-document? canvas?
- How much of shadcn/Radix is reused vs. replaced?
- Mobile/tablet treatment.

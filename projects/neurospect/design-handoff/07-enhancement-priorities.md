# NeuroSpect Marketing Site — Enhancement Priorities

## Priority 1: Flagship Visualizations (must be exceptional)

These are the "wow" moments. A trader should see these and immediately understand that NeuroSpect is a serious platform.

### 1A. Equity Curve Chart
- Progressive line draw animation (left to right)
- Pulsing trade markers that invite clicks
- Smooth line toggle transitions
- Premium tooltip with crosshair
- Subtle trading-terminal background texture

### 1B. AI Coach Chat Demo
- Typing/streaming animation on AI responses
- Staggered message entrance
- Checklist items animate with pass/fail icons
- Auto-cycling through 2-3 different scenarios
- Should feel like watching a real AI respond

### 1C. Architecture Diagram
- Node hover highlights connections (dims others)
- Particle trails with comet glow effect
- Connection labels appear on hover
- Full data-flow animation sequence on button click
- Subtle parallax on mouse movement

## What NOT to Change

- Copy/content — all text is finalized (see `04-content.md`)
- Data values — all mock data is calibrated to tell the right story
- Color assignments — component-to-color mapping is canonical
- Page structure — section order is intentional
- Legal disclaimers — must remain on every page
- Dark theme — this is non-negotiable, trading platforms are dark

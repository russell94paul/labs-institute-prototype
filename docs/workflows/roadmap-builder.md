# AI Roadmap Builder Workflow

The Roadmap Builder turns a lead or opportunity into a structured plan with decisions, recommendations, simulations, tasks, dependencies, approvals, and portal assignments.

## Steps

1. Choose roadmap type
2. Select goal
3. Select artist or lineup
4. Select location and event context
5. Build lineup and set times
6. Add contract and billing assumptions
7. Choose marketing strategy
8. Generate task roadmap
9. Simulate scenarios
10. Request approval

# Research Syntheses

This directory contains Claude-generated synthesis documents created by running `docs/prompts/ingest-new-research-output.md` after adding research outputs to `local-inputs/research-inbox/`.

Each synthesis extracts:
- Key findings
- Architecture implications
- Roadmap implications
- New risks
- Decisions required
- Implementation recommendations
- Phase changes
- Open questions

## Naming Convention

`{topic-number}-{topic-slug}-synthesis.md`

Example: `01-multi-tenant-saas-synthesis.md`

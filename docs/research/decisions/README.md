# Research Decisions

This directory contains decision candidates extracted from research syntheses. Each decision file captures a choice that needs to be made before implementation.

## Decision File Format

```markdown
# Decision: [Title]

## Context
What situation requires this decision?

## Options
1. Option A — description, trade-offs
2. Option B — description, trade-offs
3. Option C — description, trade-offs

## Recommendation
Which option and why.

## Research Source
Which research topic(s) informed this decision.

## Affects
Which modules, phases, or systems are impacted.

## Status
Proposed / Under review / Approved / Rejected

## Decided by
(name, date)
```

## Naming Convention

`DEC-{number}-{slug}.md`

Example: `DEC-001-tenancy-model.md`

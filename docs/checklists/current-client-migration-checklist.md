# Current Client Migration Checklist

Inventory: client profile, source systems, API docs, pipelines, Snowflake objects, Power BI reports, semantic models, refresh schedules, credentials/access references, data model, known issues, support burden, compliance needs.

Gaps: missing credentials, rate limits, dashboard owners, metric definitions, query history, failure logs, approval contacts, must-not-break list.

Safety: baseline, parity tests, dual-run, cutover, rollback, monitoring.

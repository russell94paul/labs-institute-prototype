# Permissions Model

The product should use one platform with role-based and entity-level portal views.

## Principles

- Internal LABS users can access scoped internal dashboards.
- Artists can only access their own relevant profile, tasks, uploads, gigs, and approvals.
- Clients can only access client-facing event, contract, payment, and approval information.
- External agencies can only access explicitly shared records.
- Sensitive financials, margins, negotiation notes, and simulations are internal by default.

Tabs

Site Preview
The public marketing site — hero, KPIs, Luciid's full profile with career timeline and releases, services, "With vs Without Labs" comparison, dual CTAs.
Designs
20 themes × 10 layouts = 200 combinations. Live counters, career scrubber, AI simulation, 3D globe, interactive AI chat. Switch theme/layout from the top bar.
Admin
Edit brand, KPIs, artists, services. Changes sync to Site Preview in real-time. Save to project memory.
Roadmap
P0–P10 build phases. Three-tier architecture (site, owner portal, DJ portal). Lab-Intelligence self-improvement flow.
Apps
Founder Portal (24 features) and DJ Portal (21 features) with phase tags. Switch between founder and DJ views.
Components
Lab-Intelligence, Lab-Wiki, Chat, Analytics, Social Integration, Booking Pipeline — each with examples and metrics.
Workflows
Interactive DAG simulator. 4 founder + 6 DJ workflows with AI suggestions at every step. Optimizer, comparisons, library with feedback loop.

Things to Try

Career Scrubber: Designs → Layout A → drag the slider from 2012 to 2026. All stats animate in sync.
AI Chat: Designs → Layout A → scroll to chat → type "growth roadmap" or "what should we quote for Printworks?"
Globe: All layouts → spinning 3D globe with 17 cities and flight paths from Manchester.
Workflow Sim: Workflows → Founder → "Book a DJ" → step through all 7 steps with AI guiding each decision.
Optimizer: Workflows → set simulations to 10 → click Optimize → comparison table + charts + trade-offs.
Design Combos: Try Mercury + Mosaic, Neon Wire + Terminal, Film Reel + Film Strip, Command + Terminal.

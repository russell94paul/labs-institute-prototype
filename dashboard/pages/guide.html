Section 01
Getting Started
1
Browse the Marketing Site
See the public-facing site: hero animation, artist profile with career data, services, new releases, and calls-to-action for DJs and promoters.
Scrolling Marquee KPI Bar Artist Profile Releases Activity Feed
2
Change the Theme & Layout
Use the two dropdowns in the top-right corner. 20 themes (Chrome, Dark Pulse, Gradient, Neon, Mercury...) and 10 layouts (Hero, Split, Grid, Cinematic, Radar...). Changes apply across all tabs.
Try: Set theme to Dark Pulse and layout to C Card Grid — then switch between tabs to see the theme everywhere.
3
Edit the Content
The Admin tab lets the founder change everything: brand name, KPIs, artists, services. Edits sync to the Site Preview in real-time.
Try: Change the first KPI from 54K to 100K, then switch to Site Preview — the stat updates instantly.
Section 02
Design Exploration
4
Explore 200 Design Combinations
The Designs tab renders a full interactive preview for each theme + layout combo. Includes live ticking counters, career timeline scrubber, AI simulation panel, spinning 3D globe, and interactive chat.
20 Themes 10 Layouts Live Counters 3D Globe AI Chat
5
Drag the Career Scrubber
In the Designs tab (Layout A), find the timeline slider and drag it from 2012 to 2026. All stats, milestones, and growth percentages animate in sync.
6
Chat with the AI
In the Designs tab (Layout A), scroll to the chat at the bottom. Type a question — the AI responds using real Luciid data from Lab-Intelligence.
Try asking: What should we quote for Printworks? or growth roadmap or booking pipeline or venue recommendations

Section 04
Interactive Workflows
10
Run a Workflow Simulation
Step through a real scenario with AI guiding each decision. 4 founder workflows (Book a DJ, Onboard, Marketing Campaign, Gig Logistics) and 6 DJ workflows (Gig Prep, Social Media, Release, Apply, Monthly Review, Studio).
Recommended first run: Select Founder → Book a DJ for an Event → click Run Workflow → step through all 7 steps. At the end, save to library.
11
Try the Gig Logistics Planner
12-step workflow with a colour-coded timeline progress bar. Plan travel from Manchester to Berlin — transport, hotel, venue, set, return. AI suggests at every step.
12 Steps Timeline Bar Cost Tracking ROI Calculation
12
Run the Optimizer
Set simulations to 10, choose a KPI to optimize (Revenue, Growth, Bookings, Time, Brand), click Optimize. See ranked comparison table, bar charts, trade-offs, and a summary recommendation.
13
Build a Custom Workflow
Click the AI Builder button in the Workflows tab. Describe any goal in plain language — Lab-Intelligence generates a custom workflow you can edit, reorder, and run.
Try: Plan a tour across Berlin, Amsterdam and Paris — generates a 9-step tour workflow you can customise.
Section 05
Feedback & Requests
14
Submit Change Requests
Use the Requests tab to submit site changes, feature ideas, or bug reports. Each request is tracked with status (Open → In Progress → Done), priority, type, and a threaded discussion.

Labs Institute — Prototype Guide

How to navigate and explore the platform prototype

Overview

This prototype shows what the Labs Institute platform will look like and do. It has 6 tabs across the top bar, plus theme and layout selectors that change the entire look in real-time.

This is a working prototype — not a final product. Data is sample/seed data. The goal is to show the vision, validate the design direction, and demonstrate the platform's capabilities.

The 6 Tabs

Site Preview
The public marketing site as visitors would see it. Shows the hero marquee, KPI stats, Luciid's full artist profile (career timeline, releases, venues, festivals, streaming stats), services, "With Labs vs Without" comparison, and dual CTAs for DJs and promoters.
Designs
5 theme options × 4 layout options = 20 combinations. Each renders a full interactive preview with:
  • Live ticking KPI counters — numbers increment in real-time
  • Career scrubber — drag the slider from 2012→2026 and watch all stats animate
  • AI simulation — Lab-Intelligence types out real suggestions live
  • 3D globe — spinning globe showing every city Luciid has played, with flight paths
  • AI chat demo — type a question and get a context-aware response
Admin
The founder's content editor. Change any text (brand name, tagline, KPIs, about section) and it updates the Site Preview tab in real-time. Add/remove artists and services. Click "Save to Project Memory" to persist changes.
Roadmap
The full P0–P10 build plan showing the three-tier architecture (Marketing Site, Owner Portal, DJ Portal) and the Lab-Intelligence self-improvement flow diagram.
Apps
Feature roadmaps for the two portals. Switch between Founder Portal (24 features: dashboard, roster, booking pipeline, marketing hub, finances, AI intelligence) and DJ Portal (21 features: content, calendar, growth, chat, finances).
Components
Deep-dive into each technical component: Lab-Intelligence (AI engine), Lab-Wiki (knowledge base), Chat (AI assistant), Analytics, Social Integration, and Booking Pipeline. Each has a concrete example, metrics, and feature list.

Theme & Layout Selectors

In the top-right corner of the header bar, two dropdowns let you change the look of the entire prototype:

Themes (5 options)

  • Chrome — current labs.institute design: white, clean, chrome accents
  • Dark Pulse — black background, neon green, rave-energy feel
  • Gradient — deep purple-blue flowing gradients, futuristic
  • Brutalist — raw monospace typography, harsh contrasts, red accents
  • Glass — frosted glass cards, soft blue glows on dark background

Layouts (4 options)

  • A — Hero Marquee — scrolling text hero + vertical scroll (full interactive experience)
  • B — Split Screen — text/CTAs left, artist card + stats right
  • C — Card Grid — bento-style dashboard with charts embedded
  • D — Cinematic — oversized logo, full-bleed, dramatic

Themes apply across all tabs. Layout affects the Designs tab preview.

Interactive Features to Try

Career Scrubber: Go to Designs tab → Layout A → drag the timeline slider from left (2012) to right (2026). Watch all stats, milestone text, and growth percentage change in real-time.
AI Chat: Designs tab → Layout A → scroll to the chat at the bottom. Try typing:
  • "What should we quote for Printworks?"
  • "growth roadmap"
  • "booking pipeline"
  • "venue recommendations"
  • "strategy analysis"
Live Counters: Designs tab → watch the KPI bar numbers tick up every 2 seconds (Instagram followers, streams, SoundCloud plays).
AI Simulation: Designs tab → Layout A → the "Lab-Intelligence — Live" panel types out AI-generated suggestions automatically, cycling through opportunities, gap detection, pricing, growth tips, and insights.
Globe: All layouts → scroll down to see the spinning 3D globe with glowing dots on 17 cities Luciid has played, with flight-path arcs from Manchester.
Admin Live Edit: Go to Admin tab → change the "Stat 1" value from "54K" to "100K" → switch to Site Preview tab → the KPI bar shows "100K".

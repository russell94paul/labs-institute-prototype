Things to Try
Career Scrubber
  1. Go to Designs tab
  2. Set Layout to A Hero Scroll
  3. Scroll down to the timeline slider
  4. Drag from 2012 → 2026 — watch all stats animate
AI Chat Demo
  1. Go to Designs tab → Layout A
  2. Scroll to the chat at the bottom
  3. Type: "What should we quote for Printworks?"
  4. Also try: "growth roadmap", "booking pipeline", "venue recommendations"
Run a Workflow
  1. Go to Workflows tab
  2. Select Founder → Book a DJ for an Event
  3. Click Run Workflow
  4. Step through all 7 steps — AI guides each decision
  5. At the end, click Save to Library
Gig Logistics Planner
  1. Go to Workflows tab
  2. Select Founder → Gig Day Logistics Planner
  3. Click Run Workflow — watch the timeline progress bar
  4. 12 steps from Manchester → Berlin → gig → home
Optimizer
  1. Go to Workflows tab
  2. Set Simulations to 10
  3. Set Optimize For to Revenue
  4. Click Optimize — comparison table + charts appear
